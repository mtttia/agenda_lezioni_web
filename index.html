Agenda lezioni

Organizza al meglio la tua giornata in maniera semplice e intuitiva con agenda lezioni, scarica subito l'apk
l'app è disponibile solo per android

più pesante : download apk
meno pesante : download zip

Currently v1.1

L'app sarà presto disponibile sul play store

Come installare l'app

Dall'apk

una volta terminato il download, cercare il file "com.mtttia.agenda_lezioni.apk" nei download e cliccarci sopra per installarla

Dallo zip

una volta terminato il download, cercare il file "agenda_lezioni.zip" nei download, cliccarci sopra per estrarla, poi cercare all'interno della stessa cartella, cercare la cartella "agenda_lezioni", aprirla e infine cliccare sul file "com.mtttia.agenda_lezioni.apk" per installarlo

Quando il telefono chiederà di installarla, acconsentire alla richiesta
